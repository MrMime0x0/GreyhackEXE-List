# GreyhackEXE-List
All the programs that have a .exe at the end in greyhack.

# EXE List
<br>➜ Mail.exe(For Viewing Mail)</br>
<br>➜ EmployeesViewer.exe(For viewing the employee DB in companies(Can be used to Fire employees if you own that company))</br>
<br>➜ LogViewer.exe(To view and delete the logs on a remote system or on the local system(Can also be used to view chat logs))</br>
<br>➜ Browser.exe(To view NPC, Player Made websites)</br>
<br>➜ Map.exe(To connect to proxies from the Map.conf file)</br>
<br>➜ FileExplorer.exe(To view or click on files and foldiers on the remote or local system)</br>
<br>➜ NotePad.exe(To do a little note taking on stuff)</br>
<br>➜ Stocks.exe(To buy crypto or stock in a company or to sell crypto that players have made)</br>
<br>➜ PDFReader.exe(To view and read PDF based files on a remote or local system)</br>
<br>➜ ImageViewer.exe(To view images on a remote or local system)</br>
<br>➜ ConfigCompany.exe(To configure a company you bought from buying stocks in that company(Mostly to view the remote network of the company))</br>
<br>➜ HiringManager.exe(To hire employees in the company that you own)</br>
<br>➜ CodeEditor.exe(To make programs via Greyscript and compile the programs or view the source code of the program if the source code is aviable to view)</br>
<br>➜ Manual.exe(A manual for players to view on stuff like Getting Started, Websites & Services, Terminal Programs, Visual Programs, Scripting)</br>
<br>➜ StudentsViewer.exe(To view student records on university networks and change grades on university networks)</br>
<br>➜ Terminal.exe(To interact with a local or remote system and to remove stuff on a local or remote system)</br>
