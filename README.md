# GreyhackEXE-List
All the programs that have a .exe at the end in greyhack.

# EXE List
➜ Mail.exe(For Viewing Mail)
➜ EmployeesViewer.exe(For viewing the employee DB in companies(Can be used to Fire employees if you own that company))
➜ LogViewer.exe(To view and delete the logs on a remote system or on the local system(Can also be used to view chat logs))
➜ Browser.exe(To view NPC, Player Made websites)
➜ Map.exe(To connect to proxies from the Map.conf file)
➜ FileExplorer.exe(To view or click on files and foldiers on the remote or local system)
➜ NotePad.exe(To do a little note taking on stuff)
➜ Stocks.exe(To buy crypto or stock in a company or to sell crypto that players have made)
➜ PDFReader.exe(To view and read PDF based files on a remote or local system)
➜ ImageViewer.exe(To view images on a remote or local system)
➜ ConfigCompany.exe(To configure a company you bought from buying stocks in that company(Mostly to view the remote network of the company))
➜ HireManager.exe(To hire employees in the company that you own)
➜ CodeEditor.exe(To make programs via Greyscript and compile the programs or view the source code of the program if the source code is aviable to view)
➜ Manual.exe(A manual for players to view on stuff like Getting Started, Websites & Services, Terminal Programs, Visual Programs, Scripting)
➜ StudentsViewer.exe(To view student records on university networks and change grades on university networks)
